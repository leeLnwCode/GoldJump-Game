# Gold Jump  
### 2D JavaFX Platform Game

Gold Jump is a 2D platform game developed using **Java** and **JavaFX**.  
The project focuses on object-oriented design, scene management, UI/UX implementation, and animation handling in a desktop application environment.

This project was developed as part of a software development learning journey, emphasizing clean structure, modular programming, and interactive game design.

## Project Overview
Gold Jump is a mission-based platform game where players:
- Log in to access the game
- Select a character
- Choose difficulty levels
- Complete missions such as collecting coins or keys
- Avoid obstacles and enemies
- Reach the goal to win the level

The game integrates animation, background music, UI styling, and structured scene transitions.

2. Install JavaFX SDK
3. Configure JavaFX library path
4. Run `Main.java`
